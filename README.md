# Rohith R — Portfolio

A premium, fully interactive personal portfolio website built with **Vanilla HTML, CSS, and JavaScript**.

## 🌐 Live Preview

Open `index.html` in any modern browser or serve with:

```bash
npx http-server . -p 5500 -o
```

---

## ✨ Features

- **Dark / Light Theme** — persistent across sessions via `localStorage`
- **Glassmorphism UI** — backdrop-filter cards, neon glow borders, animated background
- **Scroll Reveal Animations** — native CSS Scroll-Driven Animations (Chrome/Edge/Safari 26+) with IntersectionObserver fallback for Firefox
- **Fully Responsive** — mobile, tablet, and desktop layouts

---

## 🚀 Interactive Projects

### 🎙️ Voice Grammar Corrector
- Web Speech API for real-time speech-to-text

### 🎮 AI Tic-Tac-Toe
